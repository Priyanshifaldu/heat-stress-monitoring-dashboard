# Heat Stress Monitoring Dashboard

A modern web-based dashboard designed to monitor heat stress conditions using wearable sensor data. The application provides real-time visualizations, risk indicators, and alerts to help identify potentially dangerous heat exposure levels and improve worker safety.

---

## Overview

Heat stress is a significant occupational health concern, especially in outdoor and industrial environments. This dashboard aggregates wearable sensor data and presents key health and environmental metrics through an intuitive interface.

The system enables users to:

- Monitor heat stress indicators
- Visualize sensor readings
- Receive heat risk alerts
- Track worker safety metrics
- Analyze environmental conditions

## Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/heat-stress-monitoring-dashboard.git
```

### Navigate to Project

```bash
cd heat-stress-monitoring-dashboard
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Application will be available at:

```text
http://localhost:5173
```

---

## Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
